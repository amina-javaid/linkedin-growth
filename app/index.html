Setup guide

Show ▾

1. Install Claude Code

Requires Node.js 18+ and an Anthropic API key.

npm install -g @anthropic-ai/claude-code

2. Launch Chrome with remote debugging

Regular Chrome won't work — it must be launched with this flag. Run this once before starting any skill.

/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome \
  --remote-debugging-port=9222 \
  --user-data-dir=/tmp/chrome-debug

Verify it's running:

curl -s http://127.0.0.1:9222/json/version

3. Add Chrome MCP to Claude Code settings

Add this to ~/.claude/settings.json:

{
  "mcpServers": {
    "chrome-devtools": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-puppeteer"],
      "env": {
        "PUPPETEER_WS_ENDPOINT": "ws://127.0.0.1:9222"
      }
    }
  }
}

4. Log into LinkedIn

In the Chrome window you just opened, go to linkedin.com and sign in. The skills need an active session to access profile data.

5. Open this repo in Claude Code

cd linkedin-growth-kit
claude

Then run the commands from Step 3 above.
